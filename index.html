koalixcrm by scaphilo

koalixcrm is an opensource django-crm

koalixcrm is a tiny and easy to use Customer-Relationship-Management Software (CRM) including an also tiny and easy to use Accounting Software. It runs on a common Webserver or on a standard personal computer without expensive installation. All you need is python, django, postgresql and fop, all availabele open source on the web....

Dependencies

django-grappelli django-filebrowser django-south
